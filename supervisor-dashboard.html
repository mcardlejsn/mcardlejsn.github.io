Supervisor Dashboard

A Windows desktop app that pulls together JSON exports from all the staff apps into a single per-resident view — without any of it touching a cloud.

Get Started → See all apps ↓

Every staff export, one per-resident view

Staff export a file at the end of shift. The supervisor loads the folder on Windows and sees everything in one place — MAR, incidents, goals, resident check-ins, and ledger — organized per resident and per home. DSP-subfolder tagging shows which staff member exported which file.

Oversight without a cloud: the data moves by files you control, not a server someone else holds. Included with the full suite.

  • Aggregates every staff app — MAR, incidents, goals, checks, ledger
  • Per-resident, per-home view — across the whole suite
  • Staff tagging — shows who exported what
  • Windows desktop — included with the suite
  • No cloud — loads the files you already control
